# Reply Templates for Feishu

## 1. 多步骤任务（≥3 步）

```
--- 开始任务 ---
1. ✅ 步骤一：xxx
2. ⏳ 步骤二：xxx
3. ⬜ 步骤三：xxx
```

最终汇总格式：`2/17 → 17/17 ✅`

## 2. 验证循环

```
**Root Cause** | 一句话根因
**Fix Applied** | 做了什么修改
**Verification** | 命令 → 输出
**Risks** | 边界情况和风险
```

## 3. Before / After 对比

```
响应时间: 3s → 0.5s ✅
内存占用: 2GB → 512MB ✅
```

## 4. 调研总结

```
- 方案A: ❌ 原因...
- 方案B: ✅ 推荐，原因...
```

## 5. 安全表格（避免"一表毁所有"）

当需要在飞书回复中展示表格时，**不要**把表格和长文本放在同一条消息中：

```
-- 状态报告 --
| 渠道 | 状态 |
|------|------|
| GitHub | ✅ 正常 |
| 私聊 | ⏳ 等待 |

（下一条消息发送详细说明...）
```

## 6. 进度状态条

```
[■■■■■■■■□□] 80%
处理中：正在执行第 4/5 步...
```
