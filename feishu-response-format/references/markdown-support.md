# Feishu Markdown Support Reference

飞书 `post` 消息类型支持的完整格式列表。

## 支持的格式

| 格式 | 语法 | 示例效果 |
|:----|:----|:--------:|
| **加粗** | `**text**` | **粗体文本** |
| *斜体* | `*text*` | *斜体文本* |
| 行内代码 | `` `code` `` | `code` |
| 代码块 | ` ``` ` fenced | 多行代码块 |
| [链接](url) | `[text](url)` | [飞书官网](https://feishu.cn) |
| 无序列表 | `- item` | • item |
| 有序列表 | `1. item` | 1. item |
| ~~删除线~~ | `~~text~~` | ~~删除线~~ |
| 引用 | `> text` | 引用块 |

## 不支持的格式

| 格式 | 原因 | 替代方案 |
|:----|:----|:--------|
| `#` 标题 | 飞书 `post` 无 heading 元素 | 用 **粗体** 替代 |
| `---` 分割线 | 仅 `text` 类型可用 | 纯文本消息用 `---` |
| `- [ ]` 待办 | 飞书无 checkbox 组件 | 用 Emoji `⬜ ✅` |
| `![img](url)` | 不能内联图片 | `MEDIA:/path` |
| HTML 标签 | 纯文本，无渲染 | 不用 |
| `\|` 管道符表 | 降级为 `text` | 见 SKILL.md §2.2 表格方案 |

## 关键注意

1. **管道符表和 Markdown 不可混用** — 参见 SKILL.md §4 ⚠️ P0 "一表毁所有"
2. **代码块语言标注** — 飞书支持 ` ```python ` 但高亮效果有限
3. **链接语法** — 仅支持 `[text](url)`，不支持引用式 `[text][ref]`
4. **Emoji** — 可以直接使用 Emoji 字符 ✅ ❌ ⏳ ⬜，飞书全部支持
